# bin2dec

Et lille Python-script der konverterer binære tal til decimale tal.

## Brug

Kør scriptet fra terminalen:

```bash
python3 bin2dec.py
```

Du bliver bedt om at indtaste et binært tal, og scriptet returnerer den decimale værdi.

**Eksempel:**
Indtast et binært tal: 1010
Decimal værdi: 10

## Funktion

Du kan også importere og bruge funktionen direkte i dit eget script:

```python
from bin2dec import bin2dec

bin2dec("1010")  # returnerer 10
```
